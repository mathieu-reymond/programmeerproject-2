NAME

zigbee-message-x8b

DESCRIPTION

Een classe die een zigbee bytevector van type x8b voorstelt. De opstelling van een zigbee vector (x8b) : 

   * 0 : het type (in dit geval x8b/139)
   * 1 : de frame id
   * 2-3 : 16bit adres
   * 4 : retry-count
   * 5 : delivery-status (x00 is delivered, x24 is address not found)
   * 6 : discovery-status
