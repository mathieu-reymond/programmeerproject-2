NAME

xbee-simulation

DESCRIPTION

Dit is een klasse die het gedrag van een xbee-device nadoet. De methodes van deze klasse zijn dezelfde als de methodes van de xbee library. Zigbee berichten worden op dezelfde manier behandelt als op een echt xbee toestel.
