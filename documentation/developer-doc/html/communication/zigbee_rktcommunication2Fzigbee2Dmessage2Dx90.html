NAME

zigbee-message-x90

DESCRIPTION

Een classe die een zigbee bytevector van type x90 voorstelt. De opstelling van een zigbee vector (x90) : 

   * 0 : het type (in dit geval x90/144)
   * 1-8 : 64bit adres
   * 9-10 : 16bit adres
   * 11 : recieve option
   * 12-* : de bytevector van een zigbee-instruction

  * zigbee-message
